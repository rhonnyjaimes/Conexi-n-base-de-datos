# README

## Descripción

Este proyecto es una aplicación para la gestión de eventos académicos, profesores, materias, y asociaciones entre ellos. Proporciona una interfaz para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en la base de datos.

## Requisitos Previos

- Node.js: Asegúrate de tener Node.js instalado en tu sistema. Puedes descargarlo desde https://nodejs.org/.

## Instalación

1. Clona este repositorio:

 
   git clone https://github.com/rhonnyjaimes/Conexi-n-base-de-datos
